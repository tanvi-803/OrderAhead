# 🍽️ OrderAhead

**OrderAhead** is a MySQL-based Database Management System (DBMS) project that models a smart canteen food ordering platform. It allows users to browse menus from multiple canteens, place orders, make payments, and manage favourite food items, while enabling canteens to efficiently manage their menu offerings.

The project demonstrates the complete database design lifecycle, from conceptual modeling to implementation, using Entity-Relationship (ER) diagrams, relational schema mapping, normalization, and SQL DDL.

---

## 📖 Project Overview

OrderAhead is designed to streamline the food ordering process in campuses, offices, or organizations by providing a centralized database for managing users, canteens, food items, orders, payments, and related information. The database is structured to ensure data integrity, minimize redundancy, and support efficient querying through a fully normalized schema.

---

## ✨ Features

- 👤 User registration and authentication
- 🏪 Multi-canteen support
- 🍕 Food item management
- 🍛 Cuisine and food category organization
- 🛒 Order placement with token generation
- 💳 Payment management
- ❤️ Favourite food items
- 📋 Canteen-specific menu with pricing and preparation time
